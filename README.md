### :computer: __Como funciona__ :computer:

Acesse o terminal e execute os comandos abaixo:

~~~html
(Seu diretório)\api> npm start
Executando esse comando, a API do projeto vai iniciar.
~~~

~~~html
(Seu diretório)\app> npm start
Executando esse comando, o projeto será iniciado.
~~~
